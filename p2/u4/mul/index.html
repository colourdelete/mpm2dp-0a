Multiplication of Polynomials

Multiplication of binomials follow this rule: $$s(a+b)(c+d)=ac+ad+bc+bd$$e

This can be easily remembered by using the FOIL method:

  1. Multiply the first terms: $sac$e
  2. Multiply the outer terms: $sad$e
  3. Multiply the inner terms: $sbc$e
  4. Multiply the last terms: $sbd$e
Example $$s(3x-1)(4-2x)$$e $$s12x-6x^2-4+2x$$e $$s-6x^2+14x-4$$e
