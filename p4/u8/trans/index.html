Transformations of Functions

For functions of the form $g(x)=af(x-d)+c$, constants $a$, $d$, and $c$ change the shape or location of the graph of $g(x)$.

Mapping rule: $(x, y) \rightarrow (\frac{x}{b}+h)(ay+k)$

For any function with the form: $g(x)=af(b(x-h))+k$ )

Constants:

  • $a$ vertical stretch/compression
  • $b$ horizontal stretch/compression
  • $h$ vertical translation
  • $k$ horizontal translation

Transformations: For this equation, $y=af(b(x-h))+k$ :

  • Vertical Scaling (using $a$)
    • If $|a| > 1$: vs by $|a|$.
    • Else if $0 < |a| < 1$: vc by $|a|$.
  • Vertical Reflection (using $a$)
    • If $a < 0$: reflect on x-axis.
  • Horizontal Scaling (using $b$)
    • If $|b| > 1$: hs by $\frac{1}{|b|}$.
    • Else if $0 < |b| < 1$: hc by $\frac{1}{|b|}$.
  • Horizontal Reflection (using $b$)
    • If $b < 0$: reflect on y-axis.
  • Horizontal Translation (using $h$)
    • If $h > 0$: ht $h$ right.
    • Else uf $h < 0$: ht $h$ left.
  • Vertical Translation (using $k$)
    • If $k > 0$: vt $k$ up.
    • Else uf $k < 0$: vt $k$ down.

An invariant point is a point that is unchanged by a transformation. (not exactly the same as invariants in computer science)
