Notation for Sets

Set Notation

  • $\{…\}$ defines a set.
  • $(…)$ defines a tuple.
  • $D\{…\}$ defines the domain of a set.
  • $R\{…\}$ defines the range of a set.
Example

For a relation $S{(0, 1), (2, 3), (4, 5)}$ , $D_S{0, 2, 4}$ , and $R_S{1, 3, 5}$ . This relation is a function.

Example

For a relation $S{(0, 1), (2, 1), (4, 5)}$ , $D_S{0, 2, 4}$ , and $R_S{1, 1, 5}$ . This relation is not a function ($S(0) = S(2)$).
