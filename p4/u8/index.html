Introduction to Functions

Quick Reference

Mapping rule: $(x, y) \rightarrow (\frac{x}{b}+h)(ay+k)$

Transformations for a function: $y=af(b(x-h))+k$ (or $y=af(k(x-h))+c$ )

Transformations: For this equation, $y=af(b(x-h))+k$ :

  • Vertical Scaling (using $a$)
    • If $|a| > 1$: vs by $|a|$.
    • Else if $0 < |a| < 1$: vc by $|a|$.
  • Vertical Reflection (using $a$)
    • If $a < 0$: reflect on x-axis.
  • Horizontal Scaling (using $b$)
    • If $|b| > 1$: hs by $\frac{1}{|b|}$.
    • Else if $0 < |b| < 1$: hc by $\frac{1}{|b|}$.
  • Horizontal Reflection (using $b$)
    • If $b < 0$: reflect on y-axis.
  • Horizontal Translation (using $h$)
    • If $h > 0$: ht $h$ right.
    • Else uf $h < 0$: ht $h$ left.
  • Vertical Translation (using $k$)
    • If $k > 0$: vt $k$ up.
    • Else uf $k < 0$: vt $k$ down.

NOTE: Sets (e.g. domain and range) do not have repeats (e.g. $\{1, 2\}$ ).

NOTE: expand the macro/function as well when doing the equation (i.e. $y=|x+2|$ not $g(x)=|x|, y=g(x+2)$ )
