Absolute Values

The absolute value of a value is the distance from 0 on the number line. Since distances are always positive: $|x|\in \{0, \mathbb{Z}\}$

Example To solve abosolute values, split $|x|$ to $-x$ and $+x$. $$|2x+3|\le 4$$ $$2x+3\le 4, -2x-3\le 4$$ $$2x\le 1, -2x\le 7$$ $$x\le \frac{1}{2}, x\ge \frac{-7}{2}$$ $$\frac{-7}{2}\le x\le \frac{1}{2}$$
