Radicals

A radical is an unresolved root, especially one that uses the radical symbol / radix ($\sqrt{~^~}$).

Identities

Addition: $ a\sqrt{b}+c\sqrt{b}=(a+c)\sqrt{b} $

Multiplication: $ \sqrt{a}\sqrt{b}=\sqrt{ab}, a\le 0, b\le 0 $

Division: $ \frac{\sqrt{a}}{\sqrt{b}}=\sqrt{\frac{a}{b}}, a\le 0, b\le 0 $

Convert to an exponent: $ \sqrt[a]{b}=b^\frac{1}{a} $

Rationalization

To rationalize a radical, turn the root into an exponent, and then solve for the exponent:

Example Rationalize $\frac{1}{\sqrt{12}}-\frac{1}{\sqrt[3]{27}}$: $$\frac{1}{2\sqrt{3}}-\frac{1}{3}$$ $$\frac{\sqrt{3}}{6}-\frac{1}{3}$$ $$\frac{\sqrt{3}-2}{6}$$

A conjugate is $a-b$ for $a+b$, and must satify $(a+b)(a-b)=a^2-b^2$).

To rationalize fraction with binomial denominators, multiply the denominator with its conjugate.

Example Rationalize $\frac{\sqrt{6}}{\sqrt{3}+2\sqrt{5}}$: $$\frac{\sqrt{6}(\sqrt{3}-2\sqrt{5})}{3-20}$$ $$\frac{2\sqrt{30}-3\sqrt{2}}{17}$$
